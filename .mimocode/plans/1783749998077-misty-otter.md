# WindowsPathFix Rules

ALWAYS import WindowsPathFix at the start of any PowerShell session:

```powershell
Import-Module WindowsPathFix
```

## Mandatory Rules

### Paths with Special Characters
- Use `-LiteralPath` instead of `-Path`
- Use `Test-PathSafe` instead of `Test-Path`
- Use `Get-SafePath` to get properly escaped paths

### Running Python
- Use `Invoke-PythonSafe` instead of `python -c "..."`

### Writing Files
- Use `New-TempFile` for paths with special chars or sandbox issues

### Quick Aliases
- `lcat`, `lni`, `lrm`, `lcp`, `lmv` — LiteralPath versions
```

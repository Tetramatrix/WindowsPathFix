# Plan: WindowsPathFix — Add AI Agent Helpers

## Goal
Add helper functions to WindowsPathFix that help AI agents avoid common PowerShell/Windows issues:
1. Safe execution helpers (avoid `$` and quoting issues)
2. Temp file helpers (avoid sandbox/file locking issues)
3. Better error messages

## Changes

### 1. Add Invoke-Safe function
Execute PowerShell commands safely without `$` expansion issues:
```powershell
Invoke-Safe -Command "Get-Item -LiteralPath 'D:\...\[ Projects ]'"
```

### 2. Add Invoke-PythonSafe function
Run Python scripts safely, avoiding PowerShell quoting issues:
```powershell
Invoke-PythonSafe -Script "print('hello')"
Invoke-PythonSafe -File "script.py"
```

### 3. Add New-TempFile helper
Create temp files safely, then move to target (avoids sandbox issues):
```powershell
New-TempFile -Content "data" -Target "D:\...\file.txt"
```

### 4. Add Get-SafePath helper
Get a properly escaped path for the current shell:
```powershell
$path = Get-SafePath "D:\...\[ Projects ]\file.txt"
```
